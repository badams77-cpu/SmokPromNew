The Edit Template page, collects information about a Template. The Save button updates the template if the data entered is valid, cancel returns to the Email Template Pages.

Fields in Edit Template

  • Template Name - Each template has a name, and a fixed value of this name, is used in Portal when sending messages. You can have two templates with the same name is they are in different language
  • Language - The correct language is selected from the users settings, when sending the email
  • Subject - The subject line of the email sent
  • Template Body - The body of the Email

Interpolition

If the variable exists in the system creating email, subject and body text in curly brackets {variable} is replaced by the value of the variable.

Conditionals

Between {ifVar} and {/ifVar} text in the body is on sent if the var has a value of true.
